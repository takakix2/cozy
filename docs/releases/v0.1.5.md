# cozy v0.1.5

cozy v0.1.5 is a public-facing release focused on making cozy more practical as a small `nano` alternative.

## Highlights

- Browse mode: run `cozy <directory>` or press `Ctrl+B` to open a full-screen file tree.
- Save polish: new buffers default to `untitled.txt`, relative names are anchored correctly, and collisions count up.
- Existing editor basics remain covered: open/save, search/replace, undo/redo, clipboard paste, line numbers, line wrap, syntax highlighting, and Glide mode.

## Install

```bash
cargo install cozy
```

cargo test
